# Intro to ev3.sh

####Overview
In this lesson, students will use ev3.sh to program their Lego EV3 robots.

####Learning Goals
By the end of the lesson, students will be able to
* read a function
* identify parameters to steer robot
* create a function
* read simple conditionals
